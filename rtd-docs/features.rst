Features Overview
=================

iGelGiz provides a comprehensive suite of security features to protect your WordPress website. These features are organized into logical sections within the plugin's admin interface. Below is an overview of each main feature set.

Brief Summary of Features
-------------------------

*   **General Settings (Core Content Shield):** Fundamental protections including preventing content framing, email address obfuscation, disabling RSS feeds, right-click mitigation, and controlling Pinterest pinning.
*   **Hotlink Protection:** Prevents other websites from directly linking to your media files, saving your bandwidth.
*   **Rate Limiting (Bot Detection):** Protects your site from overload by limiting the number of requests a single IP can make in a short period.
*   **User Agent Blocking:** Allows you to block specific user agents, including known bad bots and AI crawlers.
*   **Geoblocking:** Restrict access to your website from specific countries.
*   **Form Protection (Honeypot):** Protects your forms (comments, login, registration) from spam bots using invisible honeypot fields and JavaScript token validation.
*   **Activity Log:** Provides a detailed log of security events handled by the plugin, allowing you to monitor threats and plugin actions.

Navigate to the respective sections using the links above for detailed configuration instructions and best practices for each feature.
